# Cat-Theme Weather App

A modern, aesthetic, and responsive weather website with a cohesive cat theme. This app provides real-time weather data and a 5-day forecast, built with HTML, CSS, and Vanilla JavaScript. 

## Features
- **Real-Time Weather Data:** Displays current temperature, location, sky conditions, humidity, wind speed, pressure, and visibility.
- **5-Day Forecast:** Shows daily weather conditions and temperatures for the next five days.
- **Glassmorphism UI:** Features a modern, sleek interface built with CSS glassmorphism effects.
- **Dynamic Cat Visuals:** Displays cute, hand-drawn cat illustrations that dynamically match the current weather conditions.
- **Responsive Design:** Optimized for both desktop and mobile devices.
- **Additional Weather Metrics:** Discover UV index, air quality index (AQI), and dew point beautifully formatted in an elegant bento grid interface.

## Getting Started

1. Clone the repository.
2. Open `index.html` in your web browser.
3. To get live data, you will need an OpenWeatherMap API key.
4. Open `script.js` and specify your OpenWeatherMap API Key by replacing `YOUR_API_KEY`.

```js
const API_KEY = "YOUR_API_KEY";
```
